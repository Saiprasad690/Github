# Github

```
git init
```
```
git add filename
```
```
git status
```
Git diff shows what changes are made to the file
```
git diff
```
```
git commit -m "Description"
```
to know the commits id's
```
git log
```
To get back to the previous changes in the code
```
git reset hard --commit id
```
Create a new branch
```
git checkout -b name
```
switching between branches
```
git checkout name
```
